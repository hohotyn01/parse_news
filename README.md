# parse_news
This parsing occurs on the site of the correspondent.

## Install
```
1. Git clone git@github.com:hohotyn01/parse_news.git
2. Write your credential for DB in .env
3. php artisan key:generate
```

## Comand
Command: run parsing
```
php artisan parse:corespondent
```
